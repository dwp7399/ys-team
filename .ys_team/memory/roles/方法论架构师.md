# 方法论架构师 经验记忆

## 角色记忆文件路径（2026-04-15）

**错误模式**：skill 文案中硬编码英文 role-id 路径（如 `roles/spec-reviewer.md`），但本仓角色名用中文持久化，导致 full-auto 链路出现路径漂移、记忆读写失败。

**正确做法**：凡涉及角色记忆路径的 skill 文案，必须标注”本仓优先用中文文件名，英文 role-id 作为回退”；执行前先 `ls .ys_team/memory/roles/` 确认实际文件名。

**适用场景**：所有读写 `.ys_team/memory/roles/` 的环节（spec-talk、spec-work、submit、close）；本仓角色名为中文时必须遵守。
